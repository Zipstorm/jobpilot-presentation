03

The Pain — Applying to Jobs Today

The real time sink is custom questions, not name/email fields.

15-30 min per application 5-8 custom questions per form
  • Browser autofill handles name & email but can't touch open-ended questions
  • Most candidates today copy questions into ChatGPT → get mediocre, generic answers → paste back
  • This is what millions of people do today

05

Demo

Live demo on a real Greenhouse job application page.

  • Open a real Greenhouse application (e.g., Anthropic, Discord)
  • Extension detects the page → floating nudge with question count
  • Click "Auto-fill Application"
  • Side panel opens — real-time streaming progress
  • Standard fields (name, email, phone) fill instantly
  • Custom questions fill one-by-one with AI-generated, role-specific answers
  • Each field highlights as it fills — the "magic" moment
  • Done in ~30 seconds
Let the demo speak. Keep narration minimal.
